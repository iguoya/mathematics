# 行列式

# 简述

定义和七个性质

三个方法

# 七个性质

  1. ∣A=∣AT∣|A= |A^T|∣A=∣AT∣
  2. 若行列式某 行/列 全是 0,则 行列式为 0
  3. 行列式可以提取公因子 k, 到行列式外边
  4. 行列式可拆分为两个行列式之和
  5. 两行/列互换,行列式值反号
  6. 两行/列元素相等或者对应成比例,行列式为 0
  7. 某行/列 k 倍加到另一行/列,行列式值不变

# 几何法

平行四边形, 立方体, n 维空间

# 逆序数法

判断 行列式 某项 正负号
